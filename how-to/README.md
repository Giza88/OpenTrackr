# How to run OpenTrackr locally

**For markers and lecturers:** start here.

| File | Purpose |
|------|---------|
| **[START-UP-GUIDE.md](./START-UP-GUIDE.md)** | Full step-by-step instructions |
| **[start-server.ps1](./start-server.ps1)** | Windows quick start — runs server and opens browser |

## Fastest way (Windows)

1. Open this `how-to` folder.
2. Right-click **`start-server.ps1`** → **Run with PowerShell**.
3. Browse the site at [http://127.0.0.1:8765/index.html](http://127.0.0.1:8765/index.html).

The assessment website lives in **`website-development/`**, not the project root.

For troubleshooting, manual commands, and Mac/Linux steps, read **START-UP-GUIDE.md**.
